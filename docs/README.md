## Geometry Lib

### Общее описание решения
Библиотека **Geometry Lib** предназначена для работы с базовыми геометрическими фигурами:  
- круг  
- прямоугольник  
- квадрат  
- треугольник  

Реализованные функции позволяют вычислять **площадь**, **периметр** и некоторые дополнительные характеристики для конкретных типов фигур (например, диагональ или диаметр).  

Библиотека написана на языке **Python** и может быть использована как вспомогательный модуль в учебных и прикладных проектах.
### Math formulas
В этой библиотеке при разработке использовались следующие математические формулы:
#### Area
- Circle: $S = πR²$
- Rectangle: $S = ab$
- Square: $S = a²$
#### Perimeter
- Circle: $P = 2πR$
- Rectangle: $P = 2a + 2b$
- Square: $P = 4a$
#### Diagonal
- Circle (diameter): $D = 2R$
- Rectangle: $D = \sqrt{a^2 + b^2}$
- Square: $D = a\sqrt{2}$
